Qibla Time Privacy Policy

Effective Date: July 16, 2025

Last Updated: July 16, 2025

1. Information Collection and Use

Location Data

The app collects your precise location (GPS) to:

  • Calculate accurate Qibla direction
  • Determine prayer times for your location
  • Show distance to Kaaba
  • Provide city name for display purposes
Important: Location data is processed locally using Huawei Location Services and never stored on our servers.

Device Sensors

The app requires:

  • Accelerometer - For compass functionality (required feature)
  • Magnetometer - For Qibla direction calculation (required feature)

Prayer Time Data

We cache prayer times locally for:

  • Offline functionality
  • Notification scheduling
  • Countdown calculations

3. Data Sharing

We share limited information with:

  • Aladhan API - Only your coordinates to fetch prayer times (no personal identifiers)
  • Huawei Mobile Services:
    • Huawei Ads SDK
    • Huawei Push Kit for notifications
    • Huawei Location Services
No personal data is collected or shared with third parties beyond what's necessary for app functionality.

Huawei Services Data Collection

When using Huawei services, the following data may be collected by Huawei:

  • Advertising ID for personalized ads (can be reset in device settings)
  • Device information for Push Kit delivery
  • Approximate location for ad targeting

Refer to Huawei's Privacy Policy for details.

4. Data Retention

All data is processed with these principles:

  • Location data - Used temporarily and not stored beyond current session
  • Prayer times - Cached locally for offline use (cleared on app uninstall)
  • Notification preferences - Stored on device per madhhab (4 schools of thought)
  • Last known location - Stored locally to restore state between sessions
  • First run flag - Stored to set default Maliki settings once

5. User Controls

You have full control over:

  • Location Services - Disable in app or device settings
  • Prayer Notifications - Toggle individually for each prayer time
  • Madhhab Selection - Choose between 4 Islamic schools of thought
  • Huawei Ads - Reset advertising ID in device settings
  • All Data - Clear everything by uninstalling the app

Notification Controls

For each prayer (Fajr, Dhuhr, Asr, Maghrib, Isha) you can:

  • Enable/disable notifications per madhhab
  • Customize vibration patterns
  • Control azan sound playback

6. Security Measures

We implement:

  • HTTPS encryption for all API communications
  • Local storage encryption for sensitive preferences
  • Regular security updates for dependencies
  • Huawei's secure SDKs for all HMS integrations

7. Children's Privacy

Our app does not knowingly collect personal information from children under 13.

Parental Note: The app contains religious content appropriate for all ages, but we recommend parental guidance for children using location services.

8. International Data Transfers

When using Huawei services, data may be transferred to and processed in:

  • China (Huawei servers)
  • Singapore (Huawei cloud region)

These transfers are covered by Huawei's data protection agreements.
